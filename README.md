### Installation

1. Clone the repo to your local machince.

   ```sh
   git clone https://github.com/topcowmoo/dypsis-integra
   ```

2. Install NPM packages

   ```sh
   npm install
   ```

3. Start the server from the terminal

   ```sh
   npm run start
   ```
